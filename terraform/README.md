### Here we've an example of a docker_image resource

1. Clone the repository
2. Run:
    2.1 Terraform init
    2.2 Terraform plan
    2.3 Terraform apply
        2.3.1 Check the result: docker image ls | grep ubuntu
    2.4 Terraform destroy
